# Lazz Pharma

Lazz pharma is an e-commerce website which is built using React, NextJs, Typescript, Tailwind Css, Aceternity UI, NodeJs, MongoDB, Mongoose ,React-icons, JWT, SSLCommerze for payment gateway and many more.
